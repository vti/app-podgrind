=pod

=head1 NAME

App::Podmaker - manage your POD

=head1 DESCRIPTION

If a module does not have a POD it will be added (default sections
+ autogenerated public methods documentation placeholders).

If a module already has a POD it will be smartly merged leaving the
documentation, removing removed methods documentation, updating license
years etc.

=head1 VIM

Add this to your C<.vimrc> and enjoy automatic POD generation/update.

    nnoremap <silent> ,pm :%!podmaker -q<cr>
    vnoremap <silent> ,pm :!podmaker -q<cr>

=cut
